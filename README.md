#### Игроки

Реализован класс игрока с именем и балансом. Баланс может изменяться и уходить в минус

Для игроков реализована коллекция PlayerCollection, которая хранит список игроков и поддерживает индексацию, срезы и
итерацию

#### Фишки

Реализован класс Chip, который поддерживает операции изменения значения

#### Балансы казино

Для хранения балансов игроков используется CasinoBalance.
Балансы хранятся по имени игрока и любое изменение баланса логируется

#### Казино

Класс Casino объединяет игроков, гусей и балансы, а также содержит логику событий симуляции

На каждом шаге симуляции случайно выбирается одно действие:

- игрок делает ставку

- игрок выигрывает

- боевой гусь атакует игрока

- кричащий гусь уменьшает баланс всех игроков

- гусь ворует деньги у игрока

После каждого события обновляются значения в CasinoBalance, что приводит к логированию изменений балансов
